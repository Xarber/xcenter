About X-Center

X-Center is a website created by Xarber, containing all of his projects and apps. Before the creation of this website, there were lots of websites containing separate projects, like:

  • https://xarber.netlify.app
  • https://xarbot.netlify.app
  • https://xwip.netlify.app

Theese websites are still online, as the management was lost. They were also good, but I wasn't happy:
As X-Center 1.0, also theese websites were created with Mobirise, a free website creator. They had lots of cool animations and other stuff but the website wasn't mine. X-Center 1.0 was a 'container' of all the stuff i developed, so the management was a lot easier. As previously stated i did'nt like the fact the website wasn't mine, so I learnt HTML, CSS, JS to develope a NEW website, but this time, just developed by me.
So X-Center 2.0 was born, with lots of bugs but also lots of pages, and i was a little more happy than before, because it was my creation. At last X-Center 3.0 was born, introducing new pages, a LOT of optimization (Now mobile users are welcome to the site), and still mine, still leaving the same interface as the 2.0, but some visual differences. This time I had a lot of help thanks to Bootstrap, also a way to help people developing the site, but still letting them code a bit. After that one, i wanted to do something COMPLETELY mine, without ANY bug. And here we are. X-Center 4.0 is now here! This time the website was developed completely by me, with just something little from bootstrap, but not the complete structure, now the website has probably still some bugs, but that's the best I can do. I hope you will enojoy the website!
Oh, another thing, the name 'X-Center' was given because the website is, as already said, a container, so it was a true 'center' of my projects, so i added the first letter of my nickname and a divider, so it cames out 'X-Center'! However, if you want to try one of theese version, you can and will always be able to do that, but remember: in every version it's a little different. Here's how to do that:

  • 5.0 > Download 'Version Switcher' from the store
  • 4.0 > Go to version page on the sitemap
  • 3.0 > Click on 'Versions' in the navbar
  • 2.0 > Click on 'Go to version selector'
  • 1.0 > You can't do this in this version.
