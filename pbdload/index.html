What is PBDload?

PBD Load is a .pbd file loader. What does it mean? It's very simple:

You insert a .pbd file and the browser convert it to html, so it can be displayed as a website. How is this possible? PBD is a file format very similiar to HTML so with help of JS you can easily convert it to HTML. But what does PBD stand for then? PBD Stands for PageBootDocument.
