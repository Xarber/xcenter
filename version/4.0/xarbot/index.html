Xarbot PRO

Enchance your discord experience adding this bot to your server! Doing so, you will allow your users and mods to use lots of tools!

What is Xarbot Pro?

Xarbot PRO is a discord bot created by xarber, but at the start only with the intent of learn a new type of code, then it became a new way to help people, obviously not that large amount or nothing too much important but is still a thing! I hope you will enjoy the bot!

What are its commands?

    Public commands

  • x!daily = Daily reward.
  • x!xarbot-news = Bot news.
  • x!sx-m = Server-x model
  • x!dm @tag (text) = Send an DM.
  • x!bot-invite = The bot invite link.
  • x!choose a b ... = Chose an option.
  • x!sx-invite = server-x invite.
  • x!create-giveaway (item) = Create giveaway.
  • x!server-info = Server's info.
  • x!bot-info = Bot's info.
  • x!ping = XARBOT PRO's ping.

    • Premium

  • x!get-admin (code) = Get admin with code
  • x!p-dm @user (text) = Send an anonymous DM.
  • x!xarber-info = Xarber's info.
  • x!remove-bot = Bot leaves the server.

    • Economy

  • x!work = Work.
  • x!wallet = Your wallet.
  • x!store = XARBOT PRO store.
  • x!transfer = Transfer money to bank
  • x!wtransfer = Transfer money to wallet.

    • Web Search

  • x!mod_s = Search mods on mcpedl.
  • x!g_search = Google search.

    • Account

  • x!login {password} = Login on server.
  • x!logout = Logout on server.
  • x!setpassword {password} = Set an password.
  • x!resetpassword {password} = Remove password.
  • x!recoverpassword = Recover password.

    • note commands

  • x!nota = Set an note
  • x!mynotes = Your notes.
  • x!canc-notes = Delete notes.

    • Admin commands

  • x!rid (role name) = Sends role ID.
  • x!setverify (@) = Make a user verified
  • x!setunverify (@) = Make a user unverified
  • x!poll = Sends poll.
  • x!clear (n. mess.) = Delete (num.) messages.
  • x!giverole (name) = Give an role.
  • x!takerole (name) = Remove an role.
  • x!chID = current channel ID.
  • x!kick (reason) = Kick member.
  • x!ban (reason) = Ban member.
  • x!editusername (@) (name) = Edit name.
  • x!disablebot = Disable bot from server
  • x!mute = Mute an user.
  • x!unmute = Unmute an user.

I know that's a lot of commands to remember, but im sure you'll find them useful sometimes.
