X-Center Known Issues

Remember: You could not be able to access this pages in some cases. (ex: js and iframe pages)

403: Access Denied
404: Page Not Found
iFrame: Using an iFrame
JS: JavaScript Disabled
Store: Not Ready

Reasons

This issues are caused by a server or user problem:

The 404 Means the page you inserted was been moved/deleted or never existed. Check the URL and confirm you have the right link.

The 403 means you haven't got the right permissions to get inside that page. You can try again or contact help.

For the iFrame error it's pretty easy: You've been accessing the page inside another website, and that's forbidden.

The JS Error means we couldn't run JS on the website. X-Center 4.0 is highly based on JS, so you can't access the page without it.

The Store Error is shown because the store is not ready yet to be published. You can still access Smart Tools though.
