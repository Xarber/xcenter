Tools - Help

In this section you can find some answers to some questions you may ask or that has been asked trough X-Center FeedBacks.


Program Functions

Q: The tools aren't working, what could be the causes?
A: If a tool isn't working you are probably not using admin privileges or you inserted wrong parameters. Try by accepting the Admin Prompt or by checking your inputs (ex. on plugin install page, insert the disk letter WITHOUT numbers or special characters).

Q: I can't download files from the download tool. What do I do?
A: If you can't download files, there are probably 2 reasons: You don't have admin privileges OR you don't have an internet connection.

Q: How do I ask for help?
A: It's very simple, just use the X-Center Feedback Page!

Q: Some multi function tool only do one command, what do I do?
A: Some tools use the program's settings to change its functions, so, try editing the program's settings to change the tool's functions!


How to use the tools

Console: This can be useful when you don't have access to the system's console (THIS COULD BLOCK TOOLS EXECUTION!!!) and you need to run a command, you can do it from here!

System Optimization: When your system starts going bad and slow or your memory is almost full for no reason or also if network or audio services aren't working, use this tool!

File Manager: If you want to delete / install more files to this program, do it from here, this way you can free memory or gain access to new tools easily!

Get OwnerShip: If you can't edit files in a folder (such as System32), its because its TrustedInstaller's propriety. With this tool you can set the folder to yours!

Set owner to TrustedInstaller: This is the exact opposite of Get OwnerShip, this tool gives the access to a folder to TrustedInstaller.

Force File Download: When your antivirus does not allow you to download a safe file, you can use this tool to download it without being blocked by your Antivirus!

Activate Windows: If you can't or don't want to buy a Windows Activation Key, use this tool to activate your system for free, getting all of the advantages.

Toggle Windows Defender: If Windows Defender is starting to annoy you, with notifications or file removal or download blocks ecc. You can disable / enable it from here!

ISO Burner: This simply launches a program (Rufus / Etcher) to burn an ISO / IMG / OTHER IMAGES to your disk or external drive, this way you can install OSs!

Plugin Installer: When you download a plugin from File Downloader, you can install it using this tool. This way you wont use too much memory and you can be sure the plugin will install correctly!

File Downloader: This is the same of Force File Download's tool, but it lets you only download 100% safe Smart Tools's tools. (This time it compresses the files to not use too much memory).

Fix Network Services: This is one of the tools of System Optimization, but it was placed here for a faster use. (This restarts network services to try to make them work).

Check For Updates: The name is self-explanatory (When the program is out-dated you can go here to get an update and some times also new tools!).

Fetch DB: When the downloads aren't working, this can mean 2 things: You dont have internet OR the DB is out-dated (For me not updating it or you not downloading it). This tool lets you update the DB.

Restore Application: When the app starts to glitch and always give you errors, you can solve with 2 ways: 1. Updating the app (if the broken tool gets updated), 2. by using this tool and restoring the whole app.

Restart Application: When you apply your settings or you're ready for an update, you can restart the app from here (this can eventually solve some errors).

Command Executed

Your command has been sent and is now getting processed to execute it on your computer. When all the processes are finished, you will read the results here:

The exit code is 0 when a fatal error has happened during the process, 1 when it exited successfully and 2 when there was a less significant error.

When a fatal error happens, you will not be able to see the reason here. Try to press F12 or CTRL+SHIFT+I and go to the CONSOLE tab to see the errors occurred.

Result Sintax:

When the process will end (successfully or not), a notification will warn you and you will know the exit result.

You will also be able to come back in this page to get the exit code and eventually the error message
