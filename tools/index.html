Command Executed

Your command has been sent and is now getting processed to execute it on your computer. When all the processes are finished, you will read the results here:

The exit code is 0 when a fatal error has happened during the process, 1 when it exited succeffully and 2 when there was a less significant error.

When a fatal error happens, you will not be able to see the reason here. Try to press F12 or CTRL+SHIFT+I and go to the CONSOLE tab to see the errors occurred.

Result Sintax:

When the process will end (succeffully or not), a notification will warn you and you will know the exit result.

You will also be able to come back in this page to get the exit code and eventually the error message
